14.1 Changes--- 3.3

rawhtml29# Thank you to all supporters. To donate to the project, go to: http://aquamacs.org/donations.shtml.
  • Aquamacs is now based on Emacs 25 (development build).
  • The commandline tools installation function has been updated for Mac OSX ``El Capitan''. Reported by Steve Ward.
  • The Copy Formatted function (formerly Copy Styled as HTML) has been improved for compatibility with any Mac application that can insert PDF documents, including MS PowerPoint, Keynote, Pages. MS Word will see editable text. For use in Aquamacs, a HTML snippet is placed on the clipboard.
  • The color for printing text is now always black unless fontification chooses a different color. If a dark background is used, colors for black-on-white printing are always darkened. (Configure with `htmlize-white-background'.)
  • Prevent crashes when, after display of save or print panels, certain keys (enter) were pressed.
  • Python-mode updated to version 6.2.0 (was: 6.1.3). For what's new, see https://launchpad.net/python-mode/+announcement/13077.
