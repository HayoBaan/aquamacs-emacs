6.4 Startup files

In short, most settings are stored in these files:

  • <#222#>/.emacs<#222#>
  • <#223#>/Library/Preferences/Aquamacs Emacs/Preferences.el<#223#>
  • <#224#>/Library/Preferences/Aquamacs Emacs/customizations.el<#224#>

    [Automatically written when you ;SPMquot;Save Options;SPMquot;]

There are further source files that will be loaded. So, the complete answer is as follows.

At startup, Aquamacs will load this file:

  • <#227#>/.emacs<#227#>

At startup, by default it will load the following files at startup:

  • <#230#>/Library/Preferences/Aquamacs Emacs/customizations.el<#230#>
  • <#231#>/Library/Application Support/Emacs/Preferences.el<#231#>

    [deprecated: don't use!]

  • <#232#>/Library/Application Support/Aquamacs Emacs/Preferences.el<#232#>
  • <#233#>/Library/Application Support/Emacs/Preferences.el<#233#>

    [deprecated: don't use!]

  • <#234#>/Library/Application Support/Aquamacs Emacs/Preferences.el<#234#>

The location of the first customizations file is given by the `custom-file' variable.

Aquamacs will also automatically load all files names <#236#>site-start.el<#236#> in the load path list, which includes the above paths.

Note that Emacs (including Aquamacs) will preferentially load a <#237#>foo.elc<#237#> file if present, in lieu of a <#238#>foo.el<#238#> file, even if the <#239#>.el<#239#> file is newer.

To trouble-shoot your own customizations, first start Aquamacs ;SPMquot;without customizations;SPMquot;, which you can do via a function found in the menu ;SPMquot;Help/Diagnose and Report Bug;SPMquot;.
