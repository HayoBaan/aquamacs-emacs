2.9 Sending

C-c C-c

Send the message and bury the current buffer (message-send-and-exit).

C-c C-s

Send the message (message-send).

C-c C-d

Bury the message buffer and exit (message-dont-send).

C-c C-k

Kill the message buffer and exit (message-kill-buffer).
